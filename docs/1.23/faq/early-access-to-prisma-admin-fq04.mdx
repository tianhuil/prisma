## FAQ

[Prisma Admin](https://www.prisma.io/admin) is GUI-based database management tool. It provides a tabular view on your data and lets you make changes through the GUI. Prisma Admin also features a plugin system that lets you integrate custom views for your data, such as a map for location data or a color palette for color values.
