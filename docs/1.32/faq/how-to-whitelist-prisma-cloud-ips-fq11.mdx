## FAQ

[Prisma Cloud](https://app.prisma.io) is running on [AWS Lambda](https://aws.amazon.com/lambda/) in the `eu-west-1` [region](https://docs.aws.amazon.com/general/latest/gr/rande.html). To learn which IP addresses are used in that region, you can check out the [AWS docs](https://docs.aws.amazon.com/general/latest/gr/aws-ip-ranges.html) on that topic.
