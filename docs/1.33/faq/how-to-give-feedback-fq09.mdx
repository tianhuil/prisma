## FAQ

The feedback we receive from the Prisma community and other developers is one of the main drivers for the things we're working on at Prisma. This FAQ helps you find the right place to submit your feedback, report problems or share ideas.

### Prisma (CLI, Prisma client, Prisma server, ...)

**I am encountering a bug in Prisma**

Please [open an issue in the main GitHub repository](https://github.com/prisma/prisma/issues/new). 

If it's urgent, you can also reach out to our support via [`support@prisma.io`](mailto:support@prisma.io)

**I have a question about Prisma**

Please ask your question in the [Prisma forum](https://www.prisma.io/forum/).

### Prisma Cloud

**I have an issue, question or feedback about Prisma Cloud**

Please [open an issue in the Prisma Cloud feedback repository](https://github.com/prisma/prisma-cloud-feedback//issues/new). 

### Content resources (docs, blog, tutorials, ...)

**I have found a typo or another mistake in a Prisma resource**

Please [open an issue in the Prisma Content feedback repository](https://github.com/prisma/prisma-content-feedback/issues/new) or directly submit a PR to fix the issue.

**I need more info about X**

Please [open an issue in the Prisma Content feedback repository](https://github.com/prisma/prisma-content-feedback/issues/new) and explain what kind of content you are looking for.

**I am encountering a bug when using a Prisma resource**

Please [open an issue in the main GitHub repository](https://github.com/prisma/prisma/issues/new). 

### Prisma example projects

**I am encountering a bug in Prisma a Prisma example project**

Please [open an issue in the `prisma-examples` repository](https://github.com/prisma/prisma-examples/issues/new).

**I want see an example for doing X with Prisma**

Please [open an issue in the `prisma-examples` repository](https://github.com/prisma/prisma-examples/issues/new).

**I have found a typo or another mistake in the README of an example project**

Please [open an issue in the `prisma-examples` repository](https://github.com/prisma/prisma-examples/issues/new) or directly submit a PR to fix the issue (following [this](https://github.com/prisma/prisma-examples/blob/master/CONTRIBUTING.md#improving-a-readme) process).

### Prisma Admin

**I have an idea for improving Prisma Admin**

Please [share your feedback and ideas in the Prisma Admin feedback repository](https://github.com/prisma/prisma-admin-feedback/issues/new).

**I am encountering a bug in Prisma Admin**

Please [open an issue in the Prisma Admin feedback repository](https://github.com/prisma/prisma-admin-feedback/issues/new).
